説明 

認知コンポーネントグループによって変換された外部の情報

データ 

No. 名前 説明 シンボル名 型 値域
1. 左側物標距離 左側の物体までの距離 leftside_distance int
 0 ～ 32767
2. 右側物標距離 右側の物体までの距離 rightside_distance int
 0 ～ 32767
3. 左前方物標距離 前方左側の物体までの距離 front_left_distance int
 0 ～ 32767
4. 右前方物標距離 前方右側の物体までの距離 front_right_distance int
 0 ～ 32767

2.  制御情報

説明 

制御コンポーネントグループに渡すハードウェア制御情報

データ 

No. 名前 説明 シンボル名 型 値域
1. 走行指示情報 現在の走行指示情報 run_info eRunInfoType
 Forward(0)：直進
TurnRight(1)：右折
TurnLeft(2)：左折
Reverse(3)：Uターン
Stop(4)：停止
Half(5)：0.5区画前進
