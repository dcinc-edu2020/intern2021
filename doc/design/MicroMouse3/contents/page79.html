マップ作成走行

説明 

1マスずつ走行しながら、3方向の壁の有無をマップ壁情報として記憶する。

コンポーネント名 

state_draw_map

入力データ 

No. データ シンボル名
1. 前壁有無
 existence_front_wall
2. 左壁有無
 existence_left_wall
3. 右壁有無
 existence_right_wall

出力データ 

No. データ シンボル名
1. マップ作成走行指示情報
 draw_map_run_info
2. マップ壁情報
 wall_info

コンポーネント内データ 

No. シンボル名 名前 型 値域
1. my_pos_x 自車の位置(x座標) int
 0 ～ 5
2. my_pos_y 自車の位置(y座標) int
 0 ～ 5
3. my_direction 自車の向き eDirectionType
 North：北
East：東
South：南
West：西

制御関数 

1.  マップ作成走行

関数名 

draw_map

引数 

No. シンボル名 型 説明 値域
1. - void
 - -

戻り値 

名前 型 値域
- void
 -

処理概要 

迷路全体の壁の有無の情報を記憶するため、

現在位置の前壁、左壁、右壁有無を迷路のマップ壁情報として記憶し、次に進む先を決める

 

その処理のうち、最初と最後の特別処理を実施する。

通常処理は「マップ作成走行指示」関数により実施する。

処理仕様 

No. 内容
1. 現在位置が(0,0)で北向きなら、初回の動作としてマップ作成走行指示情報を"0.5マス直進"に設定する。
また、現在位置を(0,1)に更新する。
2. 現在位置が(0,0)で北向き以外なら、最後の動作としてマップ作成走行指示情報を"停止"に設定する。
3. それ以外の場合、「マップ作成走行指示」関数を呼び出す。

2.  マップ作成走行指示

関数名 

draw_map_step

引数 

No. シンボル名 型 説明 値域
1. - void
 - -

戻り値 

名前 型 値域
- void
 -

処理概要 

現在の位置のマップ壁情報を、自車の向きと3方向の壁情報を考慮して記憶し、次に進むマスを決める。

また、自車の位置と向きをそれに合わせて更新する。
